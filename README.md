# Static Website for Midterm Project


## Overview

This repository contains a collection of static web projects created for the **Algoritma Pemrograman II** course. Each project demonstrates fundamental HTML, CSS, and JavaScript concepts with a focus on clean design and responsive layouts.

## Documentation

- 📚 Detailed documentation and feature overviews are available in the **docs/** folder, published via GitHub Pages.
- Each individual project has its own **README.md** with description, features, and library dependencies.

## Features & Libraries

✨ **Features**
- Responsive design using modern CSS techniques (Flexbox, Grid, container queries).
- Interactive UI components with vanilla JavaScript.
- Glassmorphism and gradient accents for a premium look.

📦 **Libraries**
- Google Fonts (Inter)
- Font Awesome icons
- No external JavaScript frameworks – pure vanilla JS.

## Quick Start

```bash
# Clone the repository (already done)
# Serve locally
npx -y serve ./
```

Browse to `http://localhost:3000` to explore the projects.
